# Yerevan Air Quality Analysis

This project analyzes air pollution (PM2.5) levels in Yerevan using open data from ArmeniaBlackouts.org.

## Dataset
Data is provided by ArmeniaBlackouts.org and includes daily air quality and weather measurements.

Source:
https://armeniablackouts.org

## Project Structure
- Exploratory Data Analysis (EDA)
- Temporal patterns and seasonality
- Extreme pollution days analysis
- Regression model for PM2.5 prediction
