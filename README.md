# Yerevan Air Quality Analysis

This project analyzes air pollution (PM2.5) levels in Yerevan using open air-quality data.

## Dataset

The dataset is **downloaded directly within the notebook** from the official
ArmeniaBlackouts.org data repository.

No local data files are required.

**Source:**  
https://armeniablackouts.org

We acknowledge ArmeniaBlackouts.org as the data provider.

## Project Structure

- Exploratory Data Analysis (EDA)
- Temporal patterns and seasonality
- Extreme pollution days analysis
- Regression model for PM2.5 prediction
